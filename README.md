# Cr

![Java11](https://img.shields.io/badge/Java-11-red)
![Maven3](https://img.shields.io/badge/MAVEN-3-blue)

&emsp;&emsp;一个轻量级的即时通讯内核。

## 第三方依赖

- [Bson](https://bsonspec.org/)
    > 二进制序列化的数据描述与交换的通用编码格式

## 构建

&emsp;&emsp;需要Maven3、JDK11+

```
    mvn clear
    mvn package
```
